# Environment variables

- `NEXT_PUBLIC_APP_NAME`: public app name
- `NEXT_PUBLIC_DEMO_MODE`: `true` or `false`
- `GOOGLE_CLIENT_ID`: Google OAuth client ID
- `GOOGLE_CLIENT_SECRET`: Google OAuth client secret
- `GOOGLE_REDIRECT_URI`: callback URL for Google OAuth
- `SUPABASE_URL`: your Supabase project URL
- `SUPABASE_ANON_KEY`: client-safe Supabase key
- `SUPABASE_SERVICE_ROLE_KEY`: server-only Supabase key
- `OPENAI_API_KEY`: used for job summaries and interview prep
- `JOB_SYNC_API_KEY`: simple protection for manual job sync route
- `FOLLOW_UP_API_KEY`: simple protection for follow-up route
