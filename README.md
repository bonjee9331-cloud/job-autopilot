# Job Autopilot SaaS

A deployable Next.js starter for an automated job search assistant.

## What it does
- Shows a dashboard for jobs, applications, interviews, and settings
- Provides demo data immediately so the app runs without external services
- Includes API routes for health checks, job sync, follow-up runs, and interview prep
- Includes Netlify function examples for scheduled tasks

## Quick start
1. Install Node.js 20 or newer.
2. Run `npm install`
3. Copy `.env.example` to `.env.local`
4. Run `npm run dev`
5. Open `http://localhost:3000`

## Deploy to Netlify
1. Push this folder to GitHub
2. Import the repo into Netlify
3. Add the environment variables from `.env.example`
4. Deploy

## Important
This starter runs in demo mode by default. To make it live, add your Google, database, and AI keys.
